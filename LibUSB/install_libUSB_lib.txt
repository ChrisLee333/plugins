How to use
----------

1. Download latest version of libUSB (e.g. 1.0.18) from sourcforge or gitHub
2. Combile the project as static lib
3. Rename libs to libusb.lib (release) and libusbd.lib (debug)
4. Copy the libs to a single folder (e.g. lib) 
5. Copy the libusb.h file to another folter (e.g. include)
6. Run CMAKE of this file
